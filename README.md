# 血液數值圖表產生器

上傳台灣健保署「健康存摺 → 檢驗檢查結果」匯出的 HTML 檔，就能自動生成血液檢驗趨勢圖與每月數值表。

## 特色
- **免安裝、免登入**：純網頁，開啟即用。
- **隱私安全**：所有資料只存在使用者自己的瀏覽器（localStorage），不會上傳到任何伺服器。
- **自動解析**：自動抓出檢驗日期、項目、數值與參考範圍。
- **手動補充**：PDF 截圖或其他來源可用「手動新增數據」輸入。
- **事件標記**：可自行加上「打血小板／發燒／復發」等關鍵事件（垂直虛線）。
- **輸出**：每張圖可下載 PNG；資料可匯出／匯入 JSON 備份。

## 使用方式
1. 到「健康存摺」下載「檢驗檢查結果」的 HTML 檔。
2. 打開本網頁，把 HTML 檔拖進上傳區。
3. 圖表與表格自動生成。想先看範例可點「載入範例」。
