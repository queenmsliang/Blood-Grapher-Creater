血液數值圖表產生器

上傳健保署「檢驗檢查結果」HTML 檔即自動生成趨勢圖　·　免安裝、資料只存在你的瀏覽器

使用說明（點開）
  1. 健保 HTML 檔：把「檢驗檢查結果_xxx.html」拖進下方框框（可一次多個），系統會自動抓出日期、項目、數值與參考值並畫圖。
  2. PDF 截圖 / 其他：PDF 是圖片無法自動讀，請用「手動新增數據」把數值打進去（選項目、填日期與數值）。
  3. 關鍵事件：打血小板、發燒、復發等不在檢驗資料裡，用「手動新增事件」加上，圖上會出現垂直虛線標記。
  4. 存檔：資料自動存在這台裝置的瀏覽器（不會上傳到任何伺服器）。可用「匯出資料」下載 JSON 備份，或在別台裝置「匯入資料」。
  5. 圖片：每張圖右上角可下載 PNG，或用「下載全部 PNG」。
※ 本工具僅供照護紀錄與就醫溝通，判讀請以醫師為準。你的檢驗資料不會離開你的瀏覽器。
